# AWS SAP-C02 모의고사

AWS Solutions Architect Professional (SAP-C02) 자격증 준비를 위한 온라인 모의고사입니다.

🌐 **바로 가기**: [모의고사 시작하기](https://hyek98.github.io/sap-c02-quiz/)

## 📚 주요 기능

### 학습 모드
- ✅ **전체 문제 풀기**: 610개 문제 전체 학습
- ❌ **오답노트**: 틀린 문제만 다시 풀기
- 🔖 **북마크**: 중요한 문제 저장 및 메모
- 📝 **상세 해설**: 각 문제마다 구체적인 한글 해설 (평균 978자)

### 시험 모드
- 🎯 **실전 모의고사**: 65문항 랜덤 출제
- ⏱️ **90분 타이머**: 실제 시험과 동일한 시간 제한
- 📊 **점수 계산**: 1000점 만점 (750점 합격)
- 📈 **통계**: 정답률, 오답 분석

### 키보드 단축키
- `1` `2` `3` `4` `5`: A, B, C, D, E 선택지 선택
- `Enter`: 정답 확인 / 다음 문제
- `R`: 다시 풀기
- `←` `→`: 이전/다음 문제

### 데이터 동기화 ⭐
- 📤 **데이터 내보내기**: 학습 진행 상황을 JSON 파일로 저장
- 📥 **데이터 가져오기**: 다른 기기에서 학습 이어하기
- 💾 **자동 저장**: LocalStorage에 자동 저장
- 🔄 **크로스 디바이스**: 집, 사무실, 핸드폰 어디서나 학습 가능

## 🚀 사용 방법

### 1. 온라인 사용 (권장)
웹 브라우저에서 바로 접속:
```
https://hyek98.github.io/sap-c02-quiz/
```

### 2. 로컬 사용
```bash
git clone https://github.com/hyek98/sap-c02-quiz.git
cd sap-c02-quiz
open index.html
```

### 3. 데이터 동기화 (기기 간 학습 진행 상황 공유)

#### 📱 핸드폰에서 → 💻 PC로
1. 핸드폰에서 홈 화면 → **"📤 데이터 내보내기"** 클릭
2. JSON 파일 다운로드
3. PC에서 같은 사이트 접속
4. 홈 화면 → **"📥 데이터 가져오기"** 클릭
5. 다운로드한 JSON 파일 선택

#### 🏠 집 PC → 🏢 사무실 PC
1. 집에서 데이터 내보내기
2. GitHub, Google Drive, iCloud 등으로 JSON 파일 공유
3. 사무실에서 데이터 가져오기

#### 💡 팁
- **병합 모드**: 기존 데이터와 새 데이터 합치기
- **교체 모드**: 기존 데이터 삭제 후 새 데이터로 교체
- 정기적으로 백업하여 데이터 손실 방지

## 🎓 학습 전략

1. **첫 회독**: 전체 문제 풀기로 전 범위 학습
2. **복습**: 오답노트로 취약한 부분 집중 학습
3. **암기**: 북마크와 메모로 핵심 개념 정리
4. **실전 연습**: 모의고사로 시간 관리 연습
5. **최종 점검**: 오답 0개 만들기

## 💾 데이터 저장

### LocalStorage
- 브라우저에 자동 저장
- 브라우저 캐시 삭제 시 데이터 손실 가능
- **정기적으로 데이터 내보내기 권장**

### JSON 파일
- 학습 진행 상황, 북마크, 메모, 시험 기록 포함
- 안전한 곳에 백업 보관
- 버전 관리 가능
